# Plan Tehnic: Rebranding Aplicație - Nihao

## Descriere

Rebranding complet al aplicației de la "SwissO2 Trading" la "Nihao". Schimbarea include actualizarea tuturor referințelor la brand, înlocuirea logo-urilor cu variantele noi pentru light și dark theme, și actualizarea tuturor textelor și metadatelor aplicației.

## Fișiere și Funcții de Modificat

### Logo-uri și Assets

**Fișiere de copiat:**
- `docs/nihao-dark.svg` → `public/nihao-dark.svg`
- `docs/nihao-light.svg` → `public/nihao-light.svg`

**Fișiere de actualizat:**
- `public/logo.svg` - va fi înlocuit cu logo-ul Nihao (sau poate fi șters dacă folosim logo-urile separate pentru dark/light)

### Componente UI

**`src/components/Header.tsx`**
- Linia 47: Înlocuirea `<img src="/logo.svg" alt="SwissO2 Trading" />` cu un logo component care afișează condiționat `nihao-light.svg` sau `nihao-dark.svg` bazat pe tema curentă
- Folosirea hook-ului `useTheme()` (deja importat) pentru a determina tema curentă
- Actualizarea atributului `alt` la "Nihao Carbon Certificates"

**Algoritm pentru logo:**
1. Import `useTheme` din `ThemeContext` (deja importat)
2. Obținerea valorii `theme` din hook
3. Render condițional: `theme === 'dark' ? '/nihao-dark.svg' : '/nihao-light.svg'`
4. Păstrarea clasei CSS existente `h-8 w-auto` pentru consistență

## Detalii de Implementare

### Logo Component Logic

Logo-ul trebuie să fie reactiv la schimbările de temă. Implementarea va folosi hook-ul `useTheme()` existent:

```typescript
const { theme } = useTheme();
const logoPath = theme === 'dark' ? '/nihao-dark.svg' : '/nihao-light.svg';
```

Logo-ul va fi afișat în Header component, înlocuind referința statică la `/logo.svg`.

### Consistență Branding

Toate referințele la "SwissO2" trebuie înlocuite cu "Nihao" în:
- Numele aplicației (appName)
- Titluri de pagină
- Textul "About" și alte referințe în conținut
- Email-uri de contact (dacă există)
- Metadata (title, description, manifest)

### Traduceri

Actualizarea tuturor traducerilor (en, ro, zh) pentru a reflecta noul brand "Nihao Carbon Certificates" în loc de "SwissO2 Trading". Cheia de traducere `swissO2Approach` va fi redenumită în `nihaoApproach` în toate fișierele de localizare.

## Note Tehnice

- Logo-urile sunt deja create în `docs/` și trebuie copiate în `public/`
- Sistemul de temă există deja și funcționează corect (ThemeContext)
- Nu sunt necesare modificări la configurarea Tailwind sau la sistemul de temă
- Logo-ul va răspunde automat la schimbările de temă prin intermediul hook-ului `useTheme()`
- Toate modificările trebuie să păstreze consistența stilului existent și să nu afecteze funcționalitatea aplicației
