# Plan: Standardizare Temă Interfață Grafică Completă

## Descriere

Crearea unui sistem de design complet și standardizat pentru aplicație, cu tokeni de design centralizați, clase CSS standardizate, componente web reutilizabile, și integrare cu biblioteci moderne. Sistemul va permite modificări de temă într-un singur loc care se propagă automat în toată aplicația.

## Context Actual

### Stack Tehnologic Existente
- **Tailwind CSS** 3.3.5 - utilitare CSS
- **Headless UI** 1.7.17 - componente accesibile (Menu, Disclosure, Transition)
- **Heroicons** 2.0.18 - iconuri SVG
- **Chart.js** 4.4.0 - grafice
- **@tailwindcss/forms** 0.5.7 - stilizare formulare
- **React** 18.2.0 - framework UI
- **TypeScript** 5.8.3 - tipizare

### Probleme Identificate
1. **Tokeni de design fragmentați**: Valori hardcodate în componente (`#12121a`, `rgba(255,255,255,0.85)`, etc.)
2. **Lipsă standardizare**: Fiecare component folosește clase Tailwind diferite pentru același scop
3. **Tema incompletă**: Dark mode există dar nu este complet standardizat
4. **Componente neconsistent**: Fiecare component definește propriile stiluri
5. **Lipsă documentație**: Nu există documentație centralizată pentru design system

## Obiective

1. Crearea unui sistem centralizat de tokeni de design
2. Standardizarea claselor CSS și utilităților
3. Definirea unei biblioteci de componente reutilizabile
4. Îmbunătățirea sistemului de teme
5. Documentație completă pentru design system

import { buttonStyles } from './Button.styles';
import { ButtonProps } from './Button.types';

export function Button({
  variant = 'primary',
  size = 'md',
  isLoading = false,
  leftIcon,
  rightIcon,
  fullWidth = false,
  className,
  children,
  disabled,
  ...props
}: ButtonProps) {
  return (
    <button
      className={buttonStyles({ variant, size, fullWidth, className })}
      disabled={disabled || isLoading}
      {...props}
    >
      {isLoading && <Spinner className="mr-2" />}
      {leftIcon && <span className="mr-2">{leftIcon}</span>}
      {children}
      {rightIcon && <span className="ml-2">{rightIcon}</span>}
    </button>
  );
}
